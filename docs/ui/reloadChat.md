# ui.reloadChat

## 描述

重载当前聊天历史界面。这会触发聊天消息的重新渲染，适用于修改了消息内容或格式（如正则脚本变更）后，需要立即看到效果的场景。

## 输入

无

## 输出

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| ok | boolean | 是否成功触发重载。 |

---

## 示例

### 代码示例 (TypeScript)

```typescript
// 修改完正则或消息后调用
await ST_API.ui.reloadChat();
```

## 响应示例

```json
{
  "ok": true
}
```
